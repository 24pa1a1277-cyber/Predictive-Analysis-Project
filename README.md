# Predictive Analytics for Real Estate Valuation

This project implements a machine learning model to forecast housing prices using historical real estate trends.

## Key Features Completed
* **Data Simulation:** Generated a structured, clean dataset modeled after housing variables (income, area population, room counts).
* **Predictive Modeling:** Implemented a **Random Forest Regressor** to identify and predict pricing patterns.
* **Evaluation & Visualization:** Calculated model accuracy using the $R^2$ score and generated a scatter plot comparing actual versus predicted pricing.

## Files Included
* `predictive_model.py`: The complete Python script containing data generation, training, and evaluation workflows.
* `predictions_plot.png`: Visual output displaying the strong correlation achieved by the model.
